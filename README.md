# Inventory Management & Replenishment System

![Dashboard Screenshot](screenshots/inventory-dashboard.png) <!-- Add your own -->

## Overview
Excel + Power BI based system for real-time inventory tracking, low-stock alerts, replenishment recommendations, demand forecasting, and optimization using Solver & What-If.

Key Features:
- INDEX-MATCH lookups for quick stock checks
- Conditional formatting for low-stock / overstock alerts
- Solver for EOQ / reorder optimization
- What-If analysis for demand scenarios
- Reorder point & safety stock calculations
- Python forecasting & advanced optimization
- Power BI dashboard with KPIs, drill-downs

Technologies: Excel (formulas, Solver, Power Query), SQL, Python (Pandas, Prophet/PuLP), Power BI.
